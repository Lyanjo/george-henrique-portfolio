Sistema Academia de Dança

Em Produção

Sobre o Projeto

Sistema completo de gerenciamento para academias de dança desenvolvido para otimizar todos os processos administrativos e operacionais. A plataforma oferece controle total sobre alunos, professores, turmas, matrículas, pagamentos, sistema de presenças com QR Code e relatórios gerenciais avançados. O sistema foi projetado com arquitetura moderna separando backend e frontend, seguindo as melhores práticas de desenvolvimento e incluindo CI/CD completo com testes automatizados.

Funcionalidades Principais

  • Gestão completa de alunos com upload de fotos e documentos
  • Portal do aluno com acesso individualizado e QR Code para check-in
  • Controle de professores com especialidades e disponibilidade
  • Gerenciamento de turmas com identificação de modalidades de dança e limite de alunos
  • Controle de matrículas e pacotes de aulas
  • Gestão financeira completa com múltiplos métodos de pagamento
  • CRM integrado com funil de vendas e gestão de leads
  • Dashboard com métricas em tempo real e calendário semanal
  • Sistema RBAC com 5 níveis de acesso
  • Autenticação JWT com proteção de rotas e tokens seguros
  • Interface totalmente responsiva
  • CI/CD completo com testes automáticos e proteção de branches
  • Sistema de notificações

Tecnologias Utilizadas

NestJS 10 TypeScript PostgreSQL TypeORM JWT Vitest React 18 Vite TailwindCSS React Router v6 ESLint GitHub Actions (CI/CD)

Status

Sistema atualmente em uso por uma academia de dança, atendendo suas necessidades diárias de gestão e proporcionando maior eficiência em todos os processos administrativos e operacionais.
