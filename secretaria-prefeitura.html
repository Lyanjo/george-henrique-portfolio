Sistema Secretaria Prefeitura

Em Produção

Sobre o Projeto

Sistema completo de gerenciamento de filas de atendimento desenvolvido para órgãos públicos, especialmente prefeituras. A plataforma oferece controle em tempo real de múltiplos departamentos simultâneos, painéis públicos para chamada de senhas e relatórios gerenciais avançados. O sistema foi projetado para melhorar a experiência do cidadão e otimizar o fluxo de atendimento presencial.

Funcionalidades Principais

  • Gestão de filas multi-departamentos (9 setores simultâneos)
  • Sistema de senhas eletrônicas (normal e preferencial)
  • Cadastro completo de munícipes com integração ViaCEP
  • Painéis públicos em tempo real para TV/monitores
  • Painéis de atendimento por departamento
  • Relatórios e dashboards avançados com 7 visualizações diferentes
  • Calendário de atendimentos com mapas de calor
  • Gráficos por horário, departamento e região
  • Exportação de dados em CSV
  • Sincronização em tempo real entre dispositivos
  • Sistema de autenticação com níveis de acesso

Tecnologias Utilizadas

HTML5 CSS3 JavaScript Supabase PostgreSQL Chart.js

Status

Sistema em produção, atendendo diariamente centenas de cidadãos e proporcionando gestão eficiente de múltiplos departamentos. A solução reduziu significativamente o tempo de espera e melhorou a organização do atendimento presencial, além de fornecer dados precisos para análises e tomada de decisão gerencial
